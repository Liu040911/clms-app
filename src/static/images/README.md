# 登录模块所需图片资源

请将以下图片放置在此目录：

## 必需图片

1. **logo.png**
   - 用途：应用 Logo（登录页面显示）
   - 建议尺寸：120x120 像素
   - 格式：PNG（支持透明背景）

2. **wechat.png**
   - 用途：微信登录图标
   - 建议尺寸：80x80 像素
   - 格式：PNG

## 可选图片

3. **login-bg.png**
   - 用途：登录页背景图（如需要）
   - 建议尺寸：750x1334 像素
   - 格式：JPG/PNG

## 当前状态

⚠️ 这些图片资源需要由设计师提供或从项目中获取。

暂时使用占位符路径，如果图片不存在，页面仍可正常显示但会显示图片加载失败。
